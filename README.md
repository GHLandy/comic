漫画汉化库
===========

本库用于放置 LCTT comic 组汉化的漫画图像，同时进行鉴评。

仓库暂由 @[GHLandy](https://github.com/GHLandy) 维护。

汉化所使用的字体，中文样式采用 _蔡云汉天真娃娃书法体_，英文样式采用 _Comic Sans MS_。
